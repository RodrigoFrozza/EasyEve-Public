# EasyEve

EasyEve is a powerful tracking and management platform for EVE Online players. It provides automated activity logging, financial tracking, and fleet management tools, all synchronized in real-time with the EVE Swagger Interface (ESI).

## Key Features

- **Automated Tracking**: Monitor Mining, Ratting, Exploration, and Abyssal activities with zero manual entry.
- **Multi-Character Dashboard**: A unified view of all your pilots, wallets, and assets.
- **Financial Intelligence**: Detailed profit/loss analysis and transaction history.
- **Fleet Management**: Tools for organizing fleets and distributing earnings fairly among members.
- **Real-time Data**: Integrated with the EVE Online Static Data Export (SDE) and live ESI API.

## Technical Showcase

This project is built using a modern full-stack architecture designed for performance and reliability:

- **Frontend**: Next.js 14 (App Router) with TypeScript.
- **Styling**: Tailwind CSS with a customized dark-themed UI (shadcn/ui).
- **Backend**: Next.js API Routes for robust server-side logic.
- **Database**: PostgreSQL with Prisma ORM for type-safe data management.
- **Authentication**: Secure OAuth2 integration with EVE Online (ESI).
- **State Management**: Zustand for efficient client-side state handling.
